# TicTacToe Game on FPGA with testbench
A simple project about Tic Tac Toe game runs on FPGA Arty Z7 done on 2nd Semester - 1st Year.

2 Players - 3x3 Tic Tac Toe game with testbench
- The project use Verilog and Vivado.

Func: Winner or Draw Detection - How the game end - Can restart and auto switch player turn.
